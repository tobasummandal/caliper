The problem

Most scientific code
is quietly broken.
Current AI tools make it worse.

The code that designs your insulin and predicts hurricanes was written by chemists, not engineers. Off-by-one errors. Wrong units. Numerical instabilities.

Bugs that don't crash — they just produce results that look right and are wrong. 30–70% of published results can’t be reproduced. Nature has retracted papers over one-character bugs.

Cursor and Copilot were trained on web code. They confidently rewrite numerical code with wrong signs, broken indexing, hallucinated math. Helios is the correctness layer they’re missing.

What it does

Three jobs.
One tool.

Grammarly for scientific code. Sits on top of every AI tool you already use — Cursor, Copilot, Claude Code — catching the bugs they introduce, proving the math is right, routing the right kernels to the right hardware.

01

Audit

Reads your scientific code and flags the silent bugs: off-by-one errors, unit mismatches, numerical instabilities. The ones that don’t crash but produce wrong answers.

→ severity-ranked findings
02

Fix

Writes a corrected, faster version. Only the bug changes. You see a clean before / after diff and click accept.

→ minimal rewrite
03

Verify

Runs original and fix on the same synthesized inputs in sandboxed subprocesses. Proves the fix is right with numerical agreement at rtol=1e-9. Research-grade evidence, not Copilot vibes.

→ proof, not opinion
04

Route

When code matches a pattern that GPUs (or eventually quantum) do well, Helios flags it with a cost-benefit. You decide whether to send it there.

→ where the time goes
Who needs this

When the result can't be wrong
and won't crash if it is.

Drug discovery

A docking score off by a few percent moves the wrong molecule into phase I. Two years later the trial reads out negative and nobody can reconstruct why.

Cost of one silent bug: ~$200M trial

Climate models

An off-by-one in a discretization loop drops a grid cell. The mean is fine. The tail is wrong. Policy follows the tail.

Cost of one silent bug: a generation

Battery design

A unit mismatch between mAh and Ah passes review. The simulation says it is safe at 4.2 V. The pack vents at 4.0.

Cost of one silent bug: a recall

Why this is the moat
We don’t just guess.
We prove it.

Linters say “this might be wrong.” Copilot says “here’s a guess.” Helios says: here is the bug, here is the fix, here are twelve sandboxed test cases showing original and fix agree on every input that doesn’t hit the bug. Research-grade evidence, hard to copy.

OpenAI and Anthropic train on GitHub — mostly the code that runs websites. Nobody trains on the messy code that runs the physical world. Nobody owns the dataset of (broken, fixed, proven). Every Helios session writes one row. We will.
